# 퀸즈헤나 CRM 서비스 디자인 개선 기획안

## 1. 개요
현재 적용된 **"Regal Verdance"** 디자인 시스템(Material Design 3 기반, 슬레이트 그레이 톤)의 완성도를 높이고, 사용자에게 더욱 프리미엄한 경험을 제공하기 위한 고도화 기획입니다.

## 2. 주요 디자인 요소 및 적용 가이드

### ① 아이콘 시스템 고도화 (Iconography)
*   **현재**: Phosphor Icons 사용 중.
*   **개선**: 모든 아이콘에 일관된 두께(`bold` 또는 `fill`)를 적용하고, 상호작용 시 색상 변화 또는 미세한 움직임을 추가하여 직관성을 높임.

### ② 타이포그래피 미세 조정 (Typography)
*   **현재**: Pretendard(국문), Public Sans(영문/숫자) 사용 중.
*   **개선**: 숫자와 영문 지표에 **Montserrat** 또는 **Inter**를 혼용하여 데이터 가독성을 극대화함. 제목과 본문의 굵기 대비(font-weight)를 더 명확히 하여 시각적 위계를 강화함.

### ③ 유동적인 애니메이션 도입 (Dynamic Interactions)
*   **프롬프트**: "스크롤 시 섹션이 페이드업되게 하고 리스트 아이템에 Stagger 효과를 넣어줘."
*   **적용**: `framer-motion`을 사용하여 페이지 로드 시 콘텐츠가 아래에서 위로 순차적으로 나타나게 구성.

### ④ 글래스모피즘 (Glassmorphism) 레이어링
*   **프롬프트**: "스티키 헤더와 모바일 네비게이션에 글래스모피즘을 적용해줘."
*   **적용**: 배경 블러(`backdrop-blur-md`)와 미세한 투명도(70~80%)를 조합하여 상단 바가 콘텐츠 위로 떠 있는 듯한 고급스러운 층위감 구현.

### ⑤ 벤토 레이아웃 (Bento Grid) 최적화
*   **프롬프트**: "중요 지표가 강조되도록 비대칭 벤토 레이아웃을 적용해줘."
*   **적용**: 균일한 그리드를 탈피하여, 중요한 데이터(오늘의 매출, 현황 등)는 `col-span-2` 또는 `row-span-2`를 사용하여 시각적 비중을 다르게 배치.

### ⑥ 스켈레톤 UI (Seamless Loading)
*   **프롬프트**: "로딩 시 스피너 대신 구조에 맞는 스켈레톤 UI를 보여줘."
*   **적용**: 데이터 로딩 중 빈 화면이나 스피너 대신, 실제 콘텐츠와 유사한 회색 박스 애니메이션을 노출하여 사용자의 체감 대기 시간을 줄임.

### ⑦ 인터랙티브 디테일 (3D Tilt & Spotlight)
*   **프롬프트**: "카드 호버 시 3D 틸트와 커서를 따라다니는 스포트라이트 효과를 넣어줘."
*   **적용**: 카드 배경에 마우스 움직임을 반영하는 라이팅 효과와 미세한 각도 변화를 주어 생동감 있는 UI 완성.

### ⑧ 성과 시각화 (Counting Animation)
*   **프롬프트**: "대시보드 KPI 숫자에 카운팅 애니메이션을 넣어줘."
*   **적용**: 지표 숫자가 로드될 때 0부터 실제 값까지 빠르게 올라가는 효과를 주어 성과를 시각적으로 축하함.

### ⑨ 모바일 네비게이션 (Mobile-First)
*   **프롬프트**: "모바일 환경에서 한 손 조작이 편하도록 하단 바 또는 명확한 햄버거 버튼을 적용해줘."
*   **적용한 상태**: 현재 사이드바의 모바일 대응 상태를 점검하고, 하단 탭 바 이동 등 모바일 전용 UI 고도화.

### ⑩ 접근성 및 SEO 강화 (Accessibility)
*   **프롬프트**: "시멘틱 태그 사용과 색상 대비를 최적화하고 키보드 내비게이션을 지원해줘."
*   **적용**: `aria-label` 추가, 포커스 링 디자인 개선, SEO를 위한 시멘틱 태그(`article`, `section`, `header`) 구조 정밀화.

## 3. 종합 개선 실행 프롬프트 (전달용)

> **"Regal Verdance 디자인 시스템의 정체성을 유지하면서, 위 10가지 기획 요소를 종합적으로 적용해줘. 특히 AI스러운 전형적인 패턴을 피하고, 실제 고급 브랜드 웹사이트에서 사용하는 디자이너 기법(Framer Motion, Skeleton UI, Glassmorphism Header)을 중심으로 화면을 고도화해줘."**
